# Health AI

## Description
A simple model fine tuning pipe line for text generation using GPT-2

## Installation

### Prerequisites
- Python 3.11
- pipenv
- pytorch


If you don't have pipenv installed, you can install it using pip:

```bash
pip install pipenv
```

```bash
git clone https://github.com/MehdiZare/healthai
cd healthai
pipenv install
```
